COME SI TRASFERISCONO LE RISORSE:
Le risorse (file,immagini,gif, ecc.) possono essere trasferite mediante le segueti metodologie:
  • URL (Uniform Resourse Locator)
  • HTML (Hyper TExt Markup Language)
  • HTTP (Hyper Text Transfert Protocol)
  • FTP (File Transfert Protocolo)
  • DNS (Domain Name System)


INFRASTUTTURE PAGINE WEB:
Una infrastrutta di una pagina web si suddivide in due parti:
  • Client side (ovvero interpreato da parte del Browser, Funziona anche senza internet)
  • Server side (ovvero tutto o in parte del documento viene gestito tramite server che poi raggiungera` il cliente con le informazione che richiede)
Nello specifico nel Client side i principali linguaggi di programmazione sono i seguenti:
  • HTML5 (viene utilizzato SOLO per il contenuto)
  • CSS3 (viene utilizzato SOLO per la rappresentazione)
  • JS (Javascript) (viene utilizzato SOLO per l'interazione)
    • JQuerry e` una libreria che viene supportato da JS.
Nello specifico nel Server side i principali linguaggi di programmazione sono i seguenti:
  • DB (Database)
  • Linguaggi interconnessi ai Database (PHP,ASP,CGI,Perl,Coldv Fusion)

CARATTERISTICHE DOMINIO:
Il dominio e` il nome e l'indirizzo di un sito web, ma è anche un modo per identificare in maniera più intuitiva i siti su Internet.
Il dominio e` caratterizzato dalla suddivisione a livelli.
  • Il dominio di I° livello e` l'ultima parde del nome di un dominio Internet.
    • Uno dei piu` utilizzati a livello nazionale e` il ".it" oppure il ".edu" nelle istituzioni scolastiche.

  • Il dominio di II° livello e` identifica il soggetto/prodotto/brand.
    • Per esempio: www.amazon.it, amazon e` il dominio di II° Livello.

  • Il dominio di III° livello identifica una parte o sezione d'interesse del dominio.
    • Per esempio: blog.pippobarilla.com oppure faq.pippobarilla.com

COLORI:

I colori si possono distinguere grazie alla tipologia RGB (Red Green Blue)
Un sistema di classificazione sono gli Esadecimali.
I codici colore esadecimali sono valori che indicano al display la quantità di colore da mostrare. I valori sono un codice speciale che rappresenta i valori di colore da 0 a 255.
Se rosso, verde e blu sono tutti al minimo 0 (rappresentato come “00” nel codice), il colore espresso è il colore nero.
Se rosso, verde e blu sono tutti al massimo 255 (rappresentati come “FF” nel codice), il colore espresso è il colore bianco.
Un sito dove si possono trovare le diverse palette, oppure creare le stesse è questo sito: Color spire


CLASSE:

L'attributo CLASS attribuire ad alcuni elementi della pagina WEB per creare gruppi e poterli modificare allo stesso tempo.
Non possono essere composti esclusivamente da un numero.
Nel foglio di stile (CSS) viene utilizzato la riga .nomeattributo{caratteristiche CSS}, che rappresenta la classe.
Nell'esempio qui proposto qui affianco viene presentato un foglio di stile integrato nella pagina web con le sue caratteristiche.
Nella sezione BODY, sono presenti dei DIV, cioè raggrupamento di una sezione di codice, con attributo "Class".

ID:

L'attributo ID serve a indicare per un SOLO ed UNICO oggetto quella determinata caratteristica che si vuole applicare.
Quest'ultimo risulta Univoco e nel foglio di stile verrà utilizzato il carattere #.
Anche questo come l'attributo CLASS andrà inserito dentro a una qualsiasi TAG nel proprio progetto HTML.
